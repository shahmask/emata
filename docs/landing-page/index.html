Designed For Every Workspace

Tailored interfaces for both everyday utility and intense AI development.

🟢 For Novices High Utility

  • Extreme Quota Protection: Heavy developer agents like **Antigravity** (especially *Antigravity Flash Medium*) can incinerate your usage quotas with complex, expensive background reasoning loops. EMATA protects your API usage by letting you run extremely cost-effective, high-reasoning models like **Gemini 3 Flash Preview**, **Gemini 3.1 Flash-Lite**, and **Gemini 3.5** with **low-thinking budgets** (which are completely unavailable in Antigravity).
  • Drop-in `geminicli` Replacement: With `geminicli` facing deprecation, EMATA serves as the ultimate drop-in shell companion. You keep your fast, lightweight terminal queries while gaining indestructible sessions and workspace isolation.
  • Google & API Token Auth: Frictionless onboarding. Authenticate instantly using your enterprise **Google Cloud Auth (gcloud / ADC)** or standard **Gemini API Key** tokens. Switch between them anytime using the :auth command.
  • Zero Context Loss: Close your laptop lid, put it to sleep, or close the terminal tab. Your context is instantly saved and resumes when you run EMATA.
  • Direct Shell Piping: Easily pipe server logs, config files, or code directly into the agent command using native shell pipelines (e.g. cat | emata).

🟣 For AI Professionals Advanced State

  • Indestructible Session Persistence: Runs on a specialized TMUX session architecture. Immune to SSH network connection drops or reboots.
  • Workspace Session Partitioning: Conversation histories are split by SHA-256 directory hashing and session names. Launch parallel sessions without overlap.
  • Precision Budget Tuning: Configure low/minimal thinking limits on preview tiers like **Gemini 3.5**, **Gemini 3.1**, and **Gemini 3.0 Flash Preview**—allowing you to run model options that are completely absent or restricted in corporate developer agents like Antigravity.
  • Dual Auth Integration: Seamlessly swaps authentication pipelines. Natively integrates with gcloud auth application-default login directly from the shell or respects fallback GEMINI_API_KEY configurations.
  • Thought-Signature Integrity: Specifically parses, flattens, and preserves complex Thought Signatures from high-reasoning models (e.g. Gemini 3.5).
  • Granular Context Hydration: Pass files (-f) or directories (-d) to inject them cleanly as parsed system prompt context at launch time.
  • Git-Diff Integration: Run with --diff to immediately feed unstaged git workspace changes to the agent as initialization context.
